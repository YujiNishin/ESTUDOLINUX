Comando para a navegação entre diretórios:
cd:
para abrir um diretório
Ex.: cd home -> abrindo pasta home

pwd:
o comando pwd irá nos mostrar onde estamos, ou seja, em qual diretório você está.

ls:
o comando ls (listar) permite exibir uma lista de arquivos em um diretório. Ele possui algumas flags (opções de comando).
Ex.: exibir os arquivos e pastar, inclusive os ocultos
    ls -la

mkdir:
o comando mkdir cria diretórios (make directory)
Ex.: mkdir dados

cp:
o comando cp (copy) é responsável por criar uma cópia
